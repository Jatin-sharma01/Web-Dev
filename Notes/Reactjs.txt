what is react js
    -react js is a popular javascript library, used for building user interface
        specially for single page applications

#Features
    - component based architecture
    - declarative syntax
    - virtual DOM
    - JSX
    - Unidirectional data flow
    - reach ecosystem
    - Ex: import React from 'react';
            function Welcome(props) {
                return <h1>Hello, {props.name}!</h1>;
                }
                export default Welcome;

    -then go to my app by using (cd my-app)
